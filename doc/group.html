Group Windows

The "New Synth Group" option under the File menu creates a new Group Window. All future synths will be placed on this window unless a newer group is created. A new Group window is not initially visible. It becomes visible once a Synth has been added to it. It is not possible to remove a group once it has been created. It is also not possible to add a synth to an older Group once a new one has been created.

SuperCollder also has an entity called a Synth Group. These are not related in any way to Llia Groups.

Groups have very little functionality, they are simply a means for the user to organize synths in windows, beyond that they do not do much. Synth icons from all groups appear on the Graph window without any distinctions.
